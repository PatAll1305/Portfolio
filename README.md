## Project Structure

The project structure is as follows:

/portfolio-website
|-- /images # Images and other media assets
|-- /views # HTML views (resume.html, achievements.html)
|-- index.css # Main CSS file
|-- index.html # Main HTML file
|-- main.js # Main JavaScript file
|-- README.md # Project README file


## Technologies Used

The website is being built using the following technologies:

- **HTML**: For the structure and content of the website.
- **CSS**: For styling and layout.
- **JavaScript**: For interactivity and dynamic content.

## Installation

To set up the project locally, follow these steps:

1. **Clone the repository:**

git clone https://github.com/PatAll1305/portfolio.git


2. **Navigate to the project directory:**

cd portfolio


No additional dependencies are required, as the project uses vanilla HTML, CSS, and JavaScript.

## Usage

To view the website locally:

1. Open `index.html` in your preferred web browser.

## Current Features

- **Home Page:** An introduction and overview of the portfolio.
- **About Page:** Information about your skills, background, and experience.
- **Projects Page:** A showcase of projects, including descriptions and links to live demos or repositories.
- **Contact Page:** A form for visitors to reach out.
